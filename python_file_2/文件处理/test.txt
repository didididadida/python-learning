</a>，初步完成了网站半成品操作，先总结如下：</p>
<h2 id="1、安装Node-js"><a href="#1、安装Node-js" class="headerlink" title="1、安装Node.js"></a>1、安装Node.js</h2><p><a target="_blank" rel="noopener" href="https://baike.baidu.com/item/node.js/7567977">Node.js | 百度百科</a></p>
<p><a href="">JavaScript | MDN</a></p>
<p>下载<a target="_blank" rel="noopener" href="https://nodejs.org/zh-cn">Node.js</a></p>
<p><code>npm install -g yarn</code></p>
<h2 id="2、安装git"><a href="#2、安装git" class="headerlink" title="2、安装git"></a>2、安装git</h2><h2 id="3、注册GitHub账号"><a href="#3、注册GitHub账号" class="headerlink" title="3、注册GitHub账号"></a>3、注册GitHub账号</h2><h2 id="4、安装hexo"><a href="#4、安装hexo" class="headerlink" title="4、安装hexo"></a>4、安装hexo</h2><h2 id="5、使用hexo主题"><a href="#5、使用hexo主题" class="headerlink" title="5、使用hexo主题"></a>5、使用hexo主题</h2><h2 id="6、生成静态文件"><a href="#6、生成静态文件" class="headerlink" title="6、生成静态文件"></a>6、生成静态文件</h2><p><code>hexo clean </code><br><br><code>hexo generate</code> # 缩写为 <code>hexo g</code></p>
<h2 id="7、与远程仓库建立关联"><a href="#7、与远程仓库建立关联" class="headerlink" title="7、与远程仓库建立关联"></a>7、与远程仓库建立关联</h2><p><code>git init </code># 初始化 Git 仓库，只需要执行一次即可<br><br><code>git checkout -b hexo</code>#新建分支</p>
<h2 id="8、部署"><a href="#8、部署" class="headerlink" title="8、部署"></a>8、部署</h2><p><code>npm install hexo-deployer-git</code>#为了更方便的部署到 GitHub Pages，Hexo 提供了 hexo-deployer-git 插件<br><br><code>hexo deploy</code>#部署</p>
<h2 id="9、备份与自动部署"><a href="#9、备份与自动部署" class="headerlink" title="9、备份与自动部署"></a>9、备份与自动部署</h2><p>#添加到缓存区<br>git add -A<br><br>git commit -m “这次做了什么更改，简单描述下即可”<br> #推送至远程仓库<br><code>git push</code><br>#第一次提交，你可能需设置一下默认提交分支#<code>git push --set-upstream origin hexo</code></p>
